## Justification des Permissions (Review Team)
*(À fournir impérativement lors de la soumission au Google Chrome Web Store)*

- **`activeTab`** : "Required to inject the typing simulation script into the specific input field that the user interacted with via the context menu. Access is strictly limited to the tab where the user explicitly triggered the extension."
- **`scripting`** : "Required to execute the typing simulation logic (`chrome.scripting.executeScript`) inside the active tab after the user clicks the context menu item."
- **`contextMenus`** : "Required to add the 'Paste with Anti-Paste Bypass' option in the browser's right-click menu, providing the core functionality of the extension."
- **`storage`** : "Required to locally save the text the user wants to paste and their preferred typing speed setting across browser sessions."

## Pratiques en matière de confidentialité (Privacy)
- **Collecte de données :** L'extension ne collecte, ne stocke et ne transmet **aucune** donnée personnelle. Le texte collé est uniquement stocké localement via l'API `chrome.storage.local` et n'est jamais envoyé sur un serveur distant.
